Minesweeper clone for Qt written in C++
=======================================

Controls:
---------

|Mouse button|Action|
|-----------|------|
|Left click|reveal a cell|
|Right click|place or remove a flag|
|Double left click (on a number)|auto reveal connected secure cells (if flags are placed right)|
